## Usage

```
obs init            - initialize configuration for websockets connection
obs server add      - add a new obs websockets server configuration
obs start           - connect to server and start logging events
obs stop            - disconnect
obs events [recent] - display last n events from log
obs scenes          - list all current scenes
obs scene switch    - switch scene
```
